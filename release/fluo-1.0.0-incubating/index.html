Fluo 1.0.0-incubating release

27 Sep 2016

Apache Fluo 1.0.0-incubating is the first release of Fluo as an Apache project and the fourth release for the project. Below are resources for this release:

  • Download release: fluo-1.0.0-incubating-bin.tgz
  • Verify release: ASC MD5 SHA1
  • Release notes
  • Documentation
  • Javadocs

Starting with 1.0.0-incubating, Apache Fluo will follow semver for all API changes. The API consists of everything under the org.apache.fluo.api package. Code outside of this package can change at any time. If your project is using Fluo code that falls outside of the API, then consider initiating a discussion about adding it to the API.

All releases can be found in the release archive
