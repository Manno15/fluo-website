Get Involved

Below are some ways that you can get involved with Apache Fluo:

Mailing lists

The dev mailing list is for general disscussion, announcing releases, organizing meet ups, etc. For discussion regarding specific issues or pull request consider using Github (Fluo, Fluo Recipes, Fluo website). The notifications and commits mailing lists receive email from automated services and can be used to observe project activity.

dev@fluo.apache.org Subscribe Unsubscribe Archive
notifications@fluo.apache.org Subscribe Unsubscribe Archive
commits@fluo.apache.org Subscribe Unsubscribe Archive

IRC

Drop by and chat about Apache Fluo at #fluo on freenode.

User feedback

Users can provide feedback by:

  • Creating an issue on our GitHub pages: Fluo, Fluo Recipes, Fluo website
  • Sending an email to our mailing list at dev@fluo.apache.org

Contributions

Contributions are welcome to all Apache Fluo projects! All projects follow a review-then-commit process. If you are interested in contributing, read our How To Contribute page.
